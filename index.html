Tentang Saya

Hai! Saya Akbar Prayuda Putra, seorang pengembang web dengan fokus utama pada backend development menggunakan bahasa PHP dan framework Laravel. Saya juga sedang belajar menjadi seorang full-stack developer dengan mempelajari frontend development. Saya lulus dari SMK Pertiwi Kuningan dengan jurusan Rekayasa Perangkat Lunak.

Saya memiliki pengalaman dalam mengerjakan proyek-proyek indie yang memberikan saya banyak kesempatan untuk mengasah keterampilan saya dalam pengembangan web. Selain itu, saya juga memiliki sertifikat PKL yang memperkuat keahlian saya di bidang ini.

Di luar pekerjaan, saya suka berolahraga dan menjalani hidup sesuai dengan moto saya: "Hidup sesuai apa yang kamu percaya". Saya selalu termotivasi untuk belajar hal baru dan terus berkembang dalam karier saya sebagai pengembang web.

Untuk melihat proyek-proyek saya, Anda dapat mengunjungi profil GitHub saya atau mengikuti saya di Instagram.
